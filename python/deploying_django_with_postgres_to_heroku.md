# Init and Deploy

On the Heroku web page:

1. Create heroku account.

1. Create heroku app.

1. Add PostgresQL addon.

1. Set up config vars on Setting tab.

On the Heroku CLI:

1. `heroku login`

1. `heroku git:remote -a project`

1. `git push heroku master`
